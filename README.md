# 🛡️ Cybersecurity Tools with Python

&gt; My first Python project! Built while learning about cybersecurity and automation.

## 📁 What's Inside

| Tool | What It Does | File |
|------|-------------|------|
| 🔍 **Port Scanner** | Checks which computer "doors" are open | `port_scanner.py` |
| 🔐 **Password Checker** | Tests password strength + shows how websites store secrets | `password_checker.py` |
| 📊 **Log Analyzer** | Reads server records and finds hackers | `log_analyzer.py` |

## 🚀 How to Run

1. Make sure you have Python installed (download from [python.org](https://python.org))
2. Download this repository (click green "Code" button → "Download ZIP")
3. Open a terminal/command prompt
4. Run any tool:
   ```bash
   python port_scanner.py
   python password_checker.py
   python log_analyzer.py
